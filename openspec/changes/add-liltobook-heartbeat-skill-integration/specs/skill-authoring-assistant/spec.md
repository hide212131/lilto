## ADDED Requirements

### Requirement: 自律スキル化前のユーザー承認
システムは、heartbeat 経由で提案されたスキル化候補を作成する前に、ユーザー承認を要求しなければならない（MUST）。

#### Scenario: 未承認では作成されない
- **WHEN** 自律スキル化候補が作成され、ユーザー承認が得られていない
- **THEN** システムはスキルを永続保存しない

#### Scenario: 承認後にのみ作成される
- **WHEN** 自律スキル化候補についてユーザーが明示的に承認する
- **THEN** システムは `skill-creator` 実行を継続してスキルを保存する

### Requirement: 類似スキルの重複作成抑止
システムは、自律スキル化候補を作成する前に既存スキルとの重複・類似を評価し、同等スキルの多重作成を抑止しなければならない（MUST）。

#### Scenario: 既存と同等の候補は抑止される
- **WHEN** スキル名または目的・手順要約が既存スキルと同等と判定される
- **THEN** システムは新規作成を実行せず、重複抑止理由を提示する
